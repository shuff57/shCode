Step 1 ✓

Every HTML page starts with some standard boilerplate code. The <!DOCTYPE html> declaration tells the browser that the document is an HTML5 file. The <html> element is the root element that encloses all the content on the page.

Example Code 
<!DOCTYPE html>
<html>

</html>

To begin, add the <!DOCTYPE html> declaration to the top of your file, followed by an opening and closing <html> tag.

Step 2 ✓

The <head> element is a container for metadata (data about the HTML document) and is placed between the <html> and <body> tags. Adding a lang="en" attribute to the opening <html> tag is important for accessibility, as it tells screen readers and browsers that the page is in English.

Example Code 
<!DOCTYPE html>
<html lang="en">
  <head>

  </head>
</html>

Add a <head> element inside the <html> element. Also, add the lang attribute with a value of en to your opening <html> tag.

Step 3 ✓

Inside the <head>, we add important metadata. The <meta charset="UTF-8"> tag specifies the character encoding for the document, ensuring that text displays correctly across different devices. The <title> element sets the title of the page, which is shown in the browser's title bar or tab.

Example Code 
<head>
  <meta charset="UTF-8">
  <title>Full Stack Curriculum</title>
</head>

Inside your <head> element, add a <meta> tag with the charset attribute set to UTF-8, and a <title> element with your desired text.

Step 4 ✓

The <body> element contains all the visible content of a webpage, like headings, paragraphs, images, and links. It comes after the closing </head> tag.

Example Code 
...
</head>
<body>

</body>
...

After your closing </head> tag, add an opening and closing <body> tag.

Step 5 ✓

The <h1> element is used for the main heading of a page. There should typically only be one <h1> per page, as it tells search engines and assistive technologies what the most important title on the page is.

Example Code 
<body>
  <h1>Welcome to freeCodeCamp</h1>
</body>

Inside your <body>, add an <h1> element with the text Welcome to freeCodeCamp.

Step 6 ✓

HTML provides six levels of headings, from <h1> to <h6>. These are used to create a hierarchical structure. An <h2> is a subheading under an <h1>.

Example Code 
<h1>Welcome to freeCodeCamp</h1>
<h2>Full Stack Curriculum</h2>

Below your <h1> element, add an <h2> element with the text Full Stack Curriculum.

Step 7 ✓

The <p> element is used for paragraphs of text. It's a fundamental block-level element for displaying most text content on a webpage.

Example Code 
<h2>Full Stack Curriculum</h2>
<p>Learn the skills to become a full stack developer</p>

Add a <p> element after your <h2> and give it the text Learn the skills to become a full stack developer.

Step 8 ✓

Now we will create the different sections of the curriculum. Each section will consist of a heading (we'll continue using <h2> for consistency) and a descriptive paragraph (<p>).

Example Code 
<h2>Introduction to HTML</h2>
<p>HTML represents the content and structure of a webpage</p>

Create the HTML section by adding an <h2> with the text Introduction to HTML followed by a <p> with the text HTML represents the content and structure of a webpage.

Step 9 ✓

Let's continue the pattern from the previous step to add the section for CSS. This reinforces the structure of the page, making it easy to read both for users and for developers looking at the code.

Example Code 
<h2>Introduction to CSS</h2>
<p>CSS is used to style a webpage</p>

Next, create the CSS section by adding an <h2> with the text Introduction to CSS followed by a <p> with the text CSS is used to style a webpage.

Step 10 ✓

To complete the content, we will add the final section for JavaScript. This follows the same structure as the HTML and CSS sections, ensuring the document is well-organized.

Example Code 
<h2>Introduction to JavaScript</h2>
<p>JavaScript adds interactivity to a webpage</p>

Finally, create the JavaScript section. Add an <h2> with the text Introduction to JavaScript followed by a <p> with the text JavaScript adds interactivity to a webpage.

Step 11 ✓

The <main> element specifies the main content of a document. It should be unique to the document and not contain content that is repeated across pages (like sidebars or navigation bars). Wrapping our content in a <main> tag improves the semantics and accessibility of the page.

Example Code 
<body>
  <main>
    <h1>Welcome to freeCodeCamp</h1>
    <h2>Full Stack Curriculum</h2>
    ...
  </main>
</body>

For better semantics and accessibility, wrap all the elements currently inside your <body> (from the <h1> to the last <p>) within a <main> element.
